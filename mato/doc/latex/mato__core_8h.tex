It is indexed by module\+\_\+id and contains G\+Arrays indexed by channel number of the particular module instance. Finally, the elements of the nested G\+Array are G\+Lists -\/ the list of all messages of the particular module in its particular channel that are still needed by the framework or any other module and thus have not been deallocated.
