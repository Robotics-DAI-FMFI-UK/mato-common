\doxysubsection{Detailed Description}
The structure describes a single individual subscription to some channel of some module. 

Each subscription is identified by a computational node unique identifier (id\+\_\+subscription) has one of the supported subscription types, pointer to a callback function of the subscriber and the module\+\_\+id of the subscriber module.
